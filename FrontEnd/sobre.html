Sobre o Sistema

KeyChart - Sistema de Chaveamento

O KeyChart é um sistema desenvolvido para facilitar a organização e o gerenciamento de torneios esportivos. Criado com o objetivo de simplificar o processo de chaveamento, o sistema permite que organizadores e administradores gerenciem competições, partidas, equipes e resultados de forma eficiente e intuitiva.

Objetivo do Projeto

O KeyChart foi desenvolvido como parte de um projeto acadêmico do Terceiro Semestre de Ciência da Computação da UNAMA. A motivação principal foi criar uma solução que facilite a organização de torneios esportivos, oferecendo um sistema de escalonamento eficiente e intuitivo para organizadores e participantes.

Tecnologias Utilizadas

  • Front-End: HTML, CSS, Bootstrap, JavaScript
  • Back-End: Python, Django, ORM
  • Banco de Dados: MySQL
  • Outras Ferramentas: Git, GitHub, Chart.js, Docker

A escolha das tecnologias foi baseada na necessidade de criar uma aplicação robusta, escalável e de fácil manutenção, garantindo uma experiência de usuário fluida e eficiente.
